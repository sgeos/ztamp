# Glossary

> **Navigation**: [Reference](./README.md) | [Documentation Root](../README.md)

Terminology and abbreviations used in this project.

| Term | Definition |
|------|------------|
| LiveView | Phoenix LiveView. A server-rendered, real-time web framework for Elixir. |
| NIF | Native Implemented Function. An Erlang/Elixir mechanism for calling native code written in languages such as Rust or C. |
| OTP | Open Telecom Platform. The Erlang standard library and design principles for building fault-tolerant applications. |
| PDF | Portable Document Format. The file format used by the government-issued job search activity form. |
| Phoenix | A web framework for the Elixir programming language. |
| TANF | Temporary Assistance for Needy Families. A United States federal assistance program. |
