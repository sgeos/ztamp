# Phase 6: Submission Tracking (V0.6)

> **Navigation**: [Roadmap](./README.md) | [Documentation Root](../README.md)

## Purpose

Track job application submissions with metadata (employer, position, date, platform, status) and generate compliance reports. Verify the 30-hour weekly job search requirement.

## Deliverables

1. Job application data model and database schema.
2. LiveView dashboard for viewing and managing applications.
3. Weekly hour tracking and compliance reporting.
4. Government form auto-population from tracked application data.
5. Export capability for completed compliance packages.

## Dependencies

- V0.5 (Submission Logic) must be complete.
- V0.2 (PDF Write PoC) and V0.3 (PDF Concatenation PoC) for form generation.

## Status

**Not started.**
