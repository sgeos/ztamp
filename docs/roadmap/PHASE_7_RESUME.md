# Phase 7: Resume Customization (V0.7)

> **Navigation**: [Roadmap](./README.md) | [Documentation Root](../README.md)

## Purpose

Customize resumes and cover letters on a per-application basis to increase submission quality and match rate. This phase encompasses multiple subtasks related to template management, content generation, and document formatting.

## Deliverables

1. Resume template management system.
2. Cover letter template management system.
3. Per-application content customization based on job posting details.
4. Document generation in PDF format.
5. Integration with submission workflow (V0.5) for attaching customized documents.

## Subtasks (to be refined)

- Base resume and cover letter template storage.
- Job posting analysis for keyword and requirement extraction.
- Template variable substitution and content tailoring.
- PDF generation from customized templates.
- Version tracking for submitted resume and cover letter variants.

## Dependencies

- V0.6 (Submission Tracking) must be complete.

**Not started.**
